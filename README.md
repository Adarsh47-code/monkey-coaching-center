# monkey-coaching-center
This is a killer homepage of monkey coaching center

Welcome to the official repository of **Monkey Coaching Center** — a clarity‑driven coaching website built to showcase toppers, courses, and a premium learning experience.

## Features
 1. **Student showcase**: Highlighting real achievers with ranks
 2. **Interactive Cards** — Smooth hover effects for a modern feel.
 3. **Signature Logo** — Playful yet premium brand identity.
 4. **Responsive Design** — Optimized for 320px mobile screens up to 1440px desktops.
 5. **WhatsApp CTA** — One‑click trial booking for parents & students.
 6. **Social Proof** — Trusted by 100+ students & parents.

## Tech Stack
1. **HTML5** - Clean, semantic structure.
2. **Responsive Layouts** — Looks amazing in all screens
3. **Custom Hover Effects** — Subtle scale + shadow polish.
